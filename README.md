# USDT-учётный бот

Бот считает за тебя:
- **кассу в рублях** и **баланс USDT** (касса бьётся после каждой сделки)
- **средний курс продажи** и **средний курс покупки** (взвешенный по объёму)
- **спред** и **реализованную прибыль**
- подсказывает по каждой сделке: ушёл в плюс или минус

## Как пользоваться

Пиши обычным текстом — бот сам распарсит:

```
Продал Гиге 475600/76.262
Продал Адахану 6500*76.5
Купил у Стефа 1020*74
Купил у Жени 300000/74.55
```

**Правило одно:**
- `*` — слева USDT, справа курс (получаем рубли)
- `/` — слева рубли, справа курс (получаем USDT)

### Команды
| Команда | Что делает |
|---|---|
| `/balance` | текущая касса и кошелёк |
| `/stats` | средние курсы, спред, прибыль |
| `/history` | последние 10 сделок |
| `/undo` | отменить последнюю сделку |
| `/setcash 4348000 1500` | задать стартовую кассу: рубли и USDT |
| `/extra 394400` | записать «Лишние ₽» |
| `/reset` | обнулить всё |
| `/help` | напоминалка |

## Установка — 3 шага

### Шаг 1. Создай бота в Telegram
1. Открой [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot`
3. Придумай имя и username (заканчивается на `_bot`)
4. **Скопируй токен** (вида `7123456789:AAH...`)

### Шаг 2. Положи токен в `.env`
1. Скопируй `.env.example` в `.env`
2. Вставь токен:
   ```
   BOT_TOKEN=7123456789:AAH...
   ```

### Шаг 3. Запусти

**На своём Mac/PC** (нужно один раз поставить Python с [python.org](https://python.org)):

```bash
pip install -r requirements.txt
python bot.py
```

Пока терминал открыт — бот онлайн. Закрыл терминал — бот ушёл спать.

## Чтобы бот работал 24/7 (бесплатно)

**Вариант 1 — Railway** (проще всего):
1. Регистрация на [railway.app](https://railway.app) через GitHub
2. Залей эти 3 файла в репозиторий GitHub
3. На Railway: New Project → Deploy from GitHub → выбери репо
4. В Variables добавь `BOT_TOKEN=твой_токен`
5. Готово. База `trades.db` сохранится в volume (включи Volume в settings)

**Вариант 2 — VPS** (если уже есть):
```bash
git clone <твой_репо> && cd usdt_bot
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
echo "BOT_TOKEN=твой_токен" > .env
nohup python bot.py &
```

## Как добавить бота в существующий чат

1. В Telegram открой чат «НАЛИЧКА / УЧЁТ»
2. Добавить участника → найди своего бота по username → добавь
3. Дай ему права читать сообщения (для не-админ-ботов в группах включи **Group Privacy = Disabled** через @BotFather: `/setprivacy` → твой бот → Disable)
4. Стартуй стартовую кассу: `/setcash 4348000 1500` и пиши сделки как обычно

Каждый чат хранит свою отдельную кассу — личка с ботом и группа считаются раздельно.

## Что под капотом
- SQLite-файл `trades.db` рядом с `bot.py` — вся история и текущее состояние
- На каждое сообщение со словами «Продал / Купил» бот ищет в нём выражение `X*Y` или `X/Y`, считает недостающее число, обновляет кассу и баланс USDT, отвечает разбором
- Средние курсы — взвешенные: `Σрублей / ΣUSDT` отдельно для продаж и покупок
